

# Phase 1 — Updated: Login-Only Auth Page

No changes to the previously approved plan except for the Auth page:

## Auth Page (`/auth`)
- Login form only: email input, password input, "Sign In" button
- No signup toggle, no "Create account" link, no registration functionality
- On success, redirect to `/` (Dashboard)
- On error, show inline error message
- Users will be manually created via Lovable Cloud / Supabase dashboard

Everything else from the approved Phase 1 plan remains the same (tables, RLS, sidebar, color scheme, placeholder pages).
